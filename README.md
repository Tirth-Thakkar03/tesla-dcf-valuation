# tesla-dcf-valuation
A complete financial model and executive summary for a Tesla DCF valuation project.
This repository contains a full Discounted Cash Flow (DCF) valuation of Tesla, Inc., including a multi‑year operating forecast, free‑cash‑flow model, cost of capital analysis, terminal value estimation, and sensitivity testing.
The goal of this project is to estimate the intrinsic equity value of Tesla based on its ability to generate future cash flows.
